# 🥗 Food Calorie Analyzer with RAG AI

An AI-powered food calorie tracking web app that gives instant 
nutrition advice using RAG (Retrieval-Augmented Generation) AI. 
Just type any food and get AI-powered nutritional insights!

## 🌟 Features
- 🔍 Search any food — Indian or international
- 🤖 AI-powered nutrition advice using RAG + Groq LLaMA
- 📊 Backed by USDA 45,000+ food database
- 🍛 Tested with sambar, banana, chicken and more!
- Clean, responsive web interface

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **AI:** Groq LLaMA (RAG - Retrieval Augmented Generation)
- **Database:** USDA 45,000+ food dataset + Indian foods CSV
- **Frontend:** HTML, CSS, JavaScript
- **Deployment:** Render
